# 简历项目描述（可直接粘贴到简历中）

## 项目名称

**简历智能优化助手** — 基于大语言模型的简历-岗位匹配分析与自动优化系统

## 项目简述（1-2行）

独立开发基于 LLM 的简历-岗位智能匹配分析工具，实现 JD 要求自动拆解、逐项评分、一键优化、面试问答预测全流程闭环，开源部署后日活用户持续增长。

## 详细描述（3-5行）

- 开发 AI 驱动的简历-岗位匹配分析系统，支持 PDF/Word/TXT 文件上传解析，自动提取教育背景、项目经验、技能关键词
- 设计**清单式评分机制**：从 JD 提取 8-15 条硬性要求逐项评估，由程序计算总分替代 LLM 直接打分，确保评分可复现、可解释，面试时可回答"如何验证准确性"
- 实现**一键优化**功能，基于分析结果自动生成优化版简历并融入 JD 关键词，实测匹配度提升 15-25 分
- 集成**面试问答预测**，根据简历+JD 自动生成 8 个面试问题及回答思路，形成"分析→优化→面试"全闭环
- 使用 Streamlit 构建交互式 Web 界面，通过 GitHub + Streamlit Cloud 实现自动化 CI/CD 部署
- 技术栈：Python, Streamlit, DeepSeek API, PyPDF2, GitHub, Streamlit Cloud

## 技术栈标签

Python / Streamlit / LLM API / Prompt Engineering / Git / Streamlit Cloud

## 面试时可展开的亮点

1. **评分一致性问题**：发现 LLM 直接打分不稳定，改为清单拆解法+程序算分
2. **产品思维**：不做简历编辑器，专注"智能匹配分析"细分方向
3. **全闭环体验**：不只分析，还产出优化版+面试问答，用户拿到结果
4. **工程化能力**：GitHub 版本控制 + Streamlit Cloud 部署
