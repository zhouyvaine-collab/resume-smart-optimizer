# 简历智能优化助手 · Resume Smart Optimizer

基于 LLM 的简历-岗位匹配分析与自动优化系统。  
**技术栈：** Python + Streamlit + DeepSeek API + PyPDF2  
**部署：** Streamlit Cloud（无需安装，浏览器打开即用）

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **简历解析** | 支持 PDF / Word / TXT 文件上传，自动提取文本 |
| **清单式匹配分析** | 从 JD 自动提取 8-15 条硬性要求，逐项✅/⚠️/❌打分 |
| **匹配度可视化** | 进度条 + 四维指标卡（技能/经验/教育/项目），分数由程序计算 |
| **一键优化** | 基于分析结果生成优化版简历，融入 JD 关键词 |
| **对比视图** | 原始 vs 优化版左右并排，高亮改动 |
| **面试问答** | 根据简历+JD 预测 8 个面试问题并给出回答思路 |
| **历史记录** | 最近 10 条分析存档，方便多岗位对比 |

---

## 与直接问 LLM 的区别

| 对比项 | 直接问 ChatGPT/DeepSeek | 本工具 |
|--------|------------------------|--------|
| 评分稳定性 | 每次分数不同，不可复现 | 清单拆解 + 程序算分，结果稳定 |
| 输入方式 | 手动粘贴 | 支持 PDF/Word 上传 |
| 输出结构 | 散装文字 | 清单 + 可视化 + 对比视图 |
| 闭环能力 | 只分析不产出 | 分析 → 优化 → 面试问答全流程 |

## 面试问答

**Q: 这个项目的难点是什么？**

A: 最大的难点是**评分一致性**。第一版让 LLM 直接打分，同一份简历两次差了 5 分。后来改成**清单拆解法**——从 JD 中提取 8-15 条具体硬性要求（技能项、经验项、教育项），由 LLM 逐项评估匹配/部分/不匹配，然后程序计算总分。这样分数可复现，而且能告诉用户具体哪一项拉了分。

**Q: 跟直接问 ChatGPT 有什么区别？**

A: 三件事：一是**输入结构化**，支持 PDF/Word 上传简历，不依赖手动粘贴；二是**输出可视化**，进度条 + 四维指标卡让用户一眼看出强弱项；三是**全闭环**，不只分析，还能一键生成优化版简历、预测面试问题。

**Q: 怎么验证优化有效？**

A: 同一份简历 + 同一个 JD，优化前分析一次拿到分数，优化后再分析一次。通过对比视图可以看到分数变化和改了什么。实际测试，优化后平均提升 15-25 分。
