# 简历智能优化助手

基于 AI 的简历-岗位匹配分析系统

## 项目简介

这是一个使用大语言模型（Claude）构建的智能简历优化工具。用户只需输入简历内容和目标岗位描述，AI 会自动分析匹配度，并给出具体的、可操作的优化建议。

## 核心功能

- **智能匹配分析**：使用 Claude AI 进行语义理解，不只是简单的关键词匹配
- **匹配度评分**：给出 0-100 的量化评分，并说明理由
- **优势识别**：找出简历中与岗位匹配的亮点
- **缺失分析**：指出简历中缺少的关键要素
- **优化建议**：给出具体的、可操作的改进建议（包括改写示例）

## 技术栈

- **前端框架**：Streamlit（Python Web 框架）
- **AI 模型**：Claude Sonnet 4（Anthropic API）
- **语言**：Python 3.8+

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

### 3. 使用说明

1. 在侧边栏输入你的 Claude API Key
2. 在左侧文本框粘贴简历内容
3. 在下方文本框粘贴目标岗位描述
4. 点击"开始分析"按钮
5. 等待 AI 分析，查看右侧的结果

## 项目特点

### 与传统简历工具的区别

- **专注匹配分析**：不做复杂的编辑器，专注于"告诉你怎么改"
- **语义理解**：使用 LLM 理解简历和岗位的深层含义，不只是关键词匹配
- **可操作建议**：不只是打分，而是给出具体的改写示例和优化方向

### 技术亮点

1. **Prompt 工程**：精心设计的分析 Prompt，确保输出结构化、可操作的建议
2. **用户体验**：简洁的双列布局，输入和结果一目了然
3. **轻量级部署**：单文件应用，可快速部署到 Streamlit Cloud

## 面试要点

**Q: 这个项目的难点是什么？**

A: 最大的难点是如何让 AI 给出"可操作的建议"，而不只是打分。我通过精心设计 Prompt，让 AI 输出包括：
- 具体到哪个项目/部分需要修改
- 提供改写示例
- 明确应该补充的关键词

例如，不是告诉用户"你缺少数据分析经验"，而是"在你的电池 SOH 项目中，补充：使用 Pandas 处理 5000+ 条时序数据，完成数据清洗和特征工程"。

**Q: 为什么不做成完整的简历编辑器？**

A: 我做了市场调研，发现编辑工具很多（Canva、简历本等），但真正能做"智能匹配分析"的少。所以我选择专注在这个点上，把它做深做透。我的目标用户是"有简历但不知道怎么针对岗位优化"的人，他们需要的是"告诉我怎么改"，而不是从零编辑。
